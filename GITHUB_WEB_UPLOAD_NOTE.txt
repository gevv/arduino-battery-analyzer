GitHub Web Upload Note

Do not upload this ZIP file as a single file if you want GitHub to display the project folders.

1. Extract this ZIP on your computer.
2. Open the extracted folder.
3. On GitHub, create a new repository.
4. Use Add file > Upload files.
5. Drag and drop the CONTENTS of this folder into GitHub.
6. Commit the upload.
